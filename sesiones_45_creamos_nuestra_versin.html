Sesiones 4-5. Creamos nuestra versión

Taller de reinterpretación musical

Opciones creativas:

Cambiar el ritmo (de lento a rápido, añadir percusión)
Modificar la instrumentación
Actualizar la letra (manteniendo el mensaje)
Fusionar con otros estilos

Herramientas digitales

Audacity: Para grabar y mezclar
Musescore: Para escribir partituras
Soundtrap: Para creación colaborativa online

Actividad práctica

Cada grupo creará un storyboard de su versión:

Versión original (30 segundos)
Versión reinterpretada (1-2 minutos)
